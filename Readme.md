## Overview

This repository contains the lecture slides for the **Foundations of Deep Learning** module, taught at [PSL-PSAI](https://psl.eu/en/education/master-s-degree-artificial-intelligence-and-society) in Paris. The course offers a rigorous, incremental introduction to deep learning—from the fundamentals of statistical learning to modern architectures such as CNNs and Transformers.

---

## 📚 Course Structure

### Part I — Deep Learning Fundamentals

| # | Lecture | Key Topics |
|:-:|---------|------------|
| 01 | Introduction | Supervised learning, loss minimisation, bias-variance trade-off, linear & non-linear models |
| 02 | Multi-Layer Perceptron | Hidden units, activation functions, output layers, canonical pairs |
| 03 | Losses | Regression & classification losses, canonical link–loss pairs |
| 04 | Gradient Descent | Batch/SGD/mini-batch, Hessian, convergence, optimisers (Momentum, RMSProp, Adam) |
| 05 | Backpropagation | Forward/backward pass, computation graph, reverse-mode autodiff |
| 06 | Training Tricks | Regularisation, vanishing/exploding gradients, training workflow |

### Part II — Architectures & Applications

| # | Lecture | Key Topics |
|:-:|---------|------------|
| 07 | CNNs / Part I | Inductive bias, computer vision, convolution operator |
| 08 | CNNs / Part II | Multi-channel convolutions, pooling, AlexNet, ResNet, transfer learning |
| 09 | Transformers / Part I | Object detection, image segmentation, natural language, word representations, autoregressive models, RNNs |
| 10 | Transformers / Part II | Attention (Query/Keys/Values), self-attention, multi-head attention, Transformer architecture, positional encoding, tokenisation |
| 11 | Group Oral Presentations | Student presentations on landmark deep learning papers |
| 12 | Natural vs Machine Intelligence | Brief history of AI & new hopes; Neuro-AI: continual learning, energy efficiency, physical world grounding |
